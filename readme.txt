== Description ==

Z-Vote is a powerful encapsulated solution for registering, administrating and cataloguing voters and their votes for your WordPress website. Please follow our installation guide and you should be up and running with your secure voting system right away!

== Installation ==

*) Upload the contents of zvote.zip to the /wp-content/plugins/zvote/directory. The final path should be /wp-content/plugins/zvote/zvote.php

*) Activate the plugin through the 'Plugins' menu in WordPress.

*) To confirm your installation, login to your Admin interface, and scroll down to the section "Plugins" and click on "Z-Vote". Now you should be presented with the Admin control panel for Z-Vote "Z-Vote WordPress Voting System".

== Frequently Asked Questions ==

= How many votes can be made from a user's I.P. address? =

Per voting subject, the client may only enter one vote per registered I.P.

= How do I add a votable entry (post or page)? =

*) Create a post in WordPress.
*) Set the post category to "contest-entry".
*) Your entries will now show the green "thumbs-up" vote-button.
